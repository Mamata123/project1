	The grep command is used to search text or searches the given file for lines containing a match to the given strings or words. By default, grep displays the matching lines. Use grep to search for lines of text that match one or many regular expressions, and outputs only the matching lines.
SYNOPSIS: 
	./project [OPTIONS] pattern [file...]

OPTIONS:
'-e'  print the whole line with pattern as highlighted. 

'-o'  print the only matching pattern.
	
'-v'  print lines which doesnt contain given pattern.
	
'-c'  print the count of matching pattern in file.
	 			
'-i'  search for the cpattern case insensitivly and print matching lines.
